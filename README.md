# 💰 Smart Finance Dashboard

## 📌 Overview
The Smart Finance Dashboard is an interactive data analytics application designed to help users track, analyze, and understand their financial behavior.  
It provides real-time insights into spending patterns, detects trends, and generates alerts to support better financial decision-making.

---

## 🚀 Features

- 🔐 User Authentication System  
- 📊 Real-time Data Visualization  
- 📈 Multi-Dataset Comparison  
- 🚨 Smart Alerts (Overspending Detection)  
- 📂 Category-wise Expense Analysis  
- 📉 Trend Analysis for Financial Planning  

---

## 🛠️ Tech Stack

- **Python**
- **Pandas** – Data manipulation & analysis  
- **Plotly** – Interactive visualizations  
- **Streamlit** – Web app framework
